The proposed project detects the drowsiness level of the driver while driving the vehicle and raises an alarm to alerts him in case the system finds him drowsy. Such an automated system will help in reducing vehicle accidents. The proposed drowsiness detection system is based on convolutional neural networks which will analyze the facial features of the driver to determine his level of drowsiness.

The main challenges are the robustness of the algorithm towards the variation of the human face and real-time processing capability. We can handle the facial variations well using conventional image processing and hand-craft computer vision algorithms. But some variations like facial expression, lighting condition, intra-class variation, and pose variation are additional issues the conventional method failed to address. An alternative solution to conventional methods is Deep Learning which provides better performance by automatically extracting and learning the significant features. Thus, we are using different Convolutional Neural Networks for handling the real-time driver drowsiness detection system. The strategy for this to be used is based on the facial behavior measures which are thoroughly exploited to characterize drowsiness.

In this project, we will train various convolutional neural networks over a given dataset. The model is trained using a data set which has many images of people with various features such as closing and opening eyes, with or without glasses, males or females, in the dark or in the light, and many more factors so that when any person is using this system in any condition, if he/she feels drowsy, the model detects the level of drowsiness and alerts the driver by beeping an alarm. Thereafter performance comparison will be done to determine the model with the best performance. The selected model will then be used for the proposed project.
